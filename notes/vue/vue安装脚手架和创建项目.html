# 1 安装node.js

根据自己的电脑，选择相应的node版本: 这是node官方下载的地址 (opens new window)

# 安装vue CLI 脚手架

安装的版本可选

这是安装脚手架的官方网站 (opens new window)

1 拉取旧版本:

npm install-g @vue/cli-init
1

2 安装新版本的命令:

npm install -g @vue/cli
1

如果在这之前 已经安装有旧版本 可以先用一下命令 将旧的版本卸载，再安装写的版本

npm uninstall vue-cli -g
1

3 查看安装的版本:

$ vue -V
@vue/cli 4.5.13
1
2
